Get started

Download InstantClick

The latest version is 3.1.0, released on 2014/12/25 (Changelog).
Production version
minified + gzipped, 2.7kb 		Development version

Other download options

The latest development version is on GitHub.

There is Bower (?): bower install instantclick

For WordPress sites, there is an unofficial WordPress plugin.

Initialize InstantClick

Include InstantClick and initialize it near the end of your pages:

1 2 3 4 5 
...
<script src="instantclick.min.js" data-no-instant></script>
<script data-no-instant>InstantClick.init();</script>
</body>
</html>

InstantClick is now activated on your website. Read those articles to get a sufficient level of proficiency with InstantClick:

  • How InstantClick works will give you the correct mental model to deal with InstantClick’s mechanism.
  • Preloading will show you the different preloading methods available.
  • Blacklisting because not all links should be preloaded.
  • Events and script re-evaluation will help you make InstantClick play nice with the rest of your JavaScript.

InstantClick isn’t always “plug and play”, you may need to adapt your site. That’s why reading the above articles is mandatory.
